Waf DotNetPad

The Waf DotNetPad is a simple and fast code editor that makes fun to program with C# or Visual Basic.

Run application »   Microsoft Store

Features

  • .NET 8.0 Code editor for C# 12 and Visual Basic 16.9.
  • Supports auto completion during programming.
  • Background compilation for immediate feedback in the Error List.
  • Provides smart syntax highlighting.
  • Try out your code with the console Output window.


Code Samples

The application comes with code samples that shows new language features of C# and Visual Basic.

  • Record (C# 9)
  • Switch Expression (C# 8)
  • Nullable Reference Types (C# 8)
  • out variables (C# 7)
  • Tuples (C# 7 / VB 15)
  • Pattern matching (C# 7)
  • Local functions (C# 7)
  • Null-conditional operator (C# 6 / VB 14)
  • nameof operator (C# 6 / VB 14)
  • Auto-property initializers & getter-only auto-properties (C# 6 / VB 14)
  • String interpolation (C# 6 / VB 14)
